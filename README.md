# Astro Guide Bot

Профессиональный телеграм-бот для создания натальных карт с проверкой точности через Swiss Ephemeris.

## Возможности

- ✅ Расчет позиций всех планет (включая Селену и Лилит)
- ✅ Проверка точности с astro.com
- ✅ Профессиональные интерпретации
- ✅ Создание натальных карт
- ✅ Поддержка системы домов Плацидуса

## Требования

- Python 3.10+
- Swiss Ephemeris (pyswisseph)
- Токен Telegram бота

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/ваш-username/astro-bot.git
cd astro-bot

# Установка зависимостей
pip install -r requirements.txt

# Создание файла .env
echo "BOT_TOKEN=ваш_токен_бота" > .env

# Запуск бота
python bot.py

Docker

# Сборка образа
docker build -t astro-bot .

# Запуск
docker run -e BOT_TOKEN="ваш_токен" astro-bot

Использование
Отправьте боту команду /start

Введите данные рождения:

Имя

Дата (ГГГГ-ММ-ДД)

Время (ЧЧ:ММ)

Город рождения

Получите профессиональный анализ

Точность расчетов
Бот автоматически проверяет точность расчетов для тестового случая astro.com:

Андрей, 25.07.1987 12:00, Ижевск

Технологии
Python Telegram Bot

Swiss Ephemeris
